# 职引·AI求职策略教练 — 设计思考

## 气质与意象

**核心比喻**：求职是一场 RPG 冒险。用户是冒险者，心仪公司是副本 Boss，投递简历是出征，面试是 Boss 战，AI 是随身战术教练。

**意象锚点**：深夜的战情指挥室——深蓝色调的屏幕光映在桌面上，橙色警示灯闪烁着行动信号，地图上标注着目标城市和公司。这是策略与热血并存的氛围：严肃但不沉重，年轻但不轻浮。

## 视觉策略

- 图形语言：圆角卡片 + 微投影 + 经验条动画 = 游戏感与专业感的平衡
- 图像方向：不用真实照片，用插画/像素风格暗示游戏化，但保持信息层级清晰
- 进度与成长：经验条、等级徽章、漏斗视图，让抽象求职过程可视化

## 配色方案

| 主色（策略/稳重） | #1A2B4C | `bg-[#1A2B4C]` | 深夜指挥室的深蓝 |

## 字体排版

- 中文优先：系统默认（-apple-system, PingFang SC, Microsoft YaHei）
- 标题：font-bold text-xl / text-lg，紧凑有力
- 正文：text-sm / text-base，行间距适中
- 数据/数字：font-mono 或 tabular-nums，对齐展示经验值、匹配分数

## 动效与交互

- 经验条获得经验时：数字跳动动画 + 进度条平滑填充
- 卡片操作：hover/tap 时微缩放 + 阴影加深
- 页面切换：fade-in，保持流畅不卡顿
- 面试房间：AI 对话气泡逐字出现（打字机效果）
- 成就解锁：全屏闪光 + 徽章弹出

## 页面结构

### 游戏驾驶舱（首页）
- 顶部：用户等级 + 称号 + 经验条（动画）
- 中部：漏斗视图（投递→面试→Offer），可点击穿透
- 底部：4 个快捷入口卡片（AI规划、简历库、我的副本、训练场）

### 职业规划沙盘
- 聊天界面：AI教练多轮对话，气泡式
- 下方：生成的职业画像卡片 + 三条路径（保底/匹配/冲刺）

### 简历库
- 列表页：简历卡片列表，每张标注版本和匹配分数
- 编辑页：文本编辑 + JD 匹配分析 + 高亮建议

### 副本大厅
- 看板视图：按状态分列（攻略中/已投递/面试中/胜利/失败）
- 详情页：JD + AI策略 + 行动按钮 + 面试历史

### 训练场（面试中心）
- 模式选择：单面/压力面/AI群面
- 面试房间：AI头像 + 对话流 + 录音/输入切换
- 报告页：雷达图 + 文字评语

### HR反向模拟
- 3份虚拟简历卡片
- 面试对话界面（用户扮演HR）
- 揭晓结果 + 招聘笔记

### 成就中心
- 成就墙：徽章网格
- 能力成长曲线
- 历史记录时间线

## 设计禁忌

- 禁止使用科技蓝+蓝紫色渐变（太AI工具化，缺少游戏感）
- 禁止全屏纯文字列表（必须有卡片/分组/视觉层级）
- 禁止空白状态无引导（必须有趣味插画+引导文案）
- 禁止按钮使用默认蓝色（必须用 #1A2B4C 深蓝或 #FF6B35 橙色）
- 禁止硬编码 px 值（使用 Tailwind 预设类）
- 禁止用手搓通用组件代替 `@/components/ui/*`
